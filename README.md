# JAVA-Metric-Imperial-Unit-Converter-with-Dark-Mode-Toggle
Simple web app to convert between metric and imperial units for length, volume, and mass. 
Enter a value, click convert, and see results instantly. 
Includes a light/dark mode toggle for easy viewing. 
Built with plain HTML, CSS, and JavaScript — clean and easy to use.

## Features

- Converts meters ↔ feet, liters ↔ gallons, kilograms ↔ pounds  
- Validates input and shows warnings for invalid values  
- Responsive toggle switch for light/dark mode  
- Clean, minimal UI with easy-to-read fonts

## Technologies Used

- HTML5  
- CSS3 (with dark mode styles)  
- Vanilla JavaScript (ES6+)

## How to Use

1. Enter a positive number in the input field  
2. Click the **Convert** button  
3. View conversion results displayed below  
4. Toggle light/dark mode using the switch in the header
